What are variables? Data types?

  • A variable is an empty space holder for a future value
  • There are a few types of values that JavaScript uses
    • Numbers
    • Strings or a sequence of characters
    • Booleans (true or false)
    • Functions (yes variables can be functions in JavaScript)
    • Arrays (lists of variables)
    • Objects

But note

  • We will go over all of these types through writing functions
  • It is extremely important to be comfortable with writing and using functions

Functions

So what is a function?

  • Think of a function as a black box, where you put things in (inputs) and out comes a result (output)
  • It's a black box because you don't necessarily need to know how it's converting the inputs
  • The parameters are the inputs, the return statement defines the output

What is a function?

Our first function

function helloWorld(first_name, last_name) {
	var message = "Hello World " + first_name + " " + last_name;
	return message;
}

var output = helloWorld("Kay", "Ashaolu");
console.log(output);

What does this function do?

What did that function do?

  • Declared a function that we put in two values or inputs: (first name and last name)
  • The function declared a variable (message) that contains the string "Hello World [your first name] [your last name]"
  • After it's done, it will produce one value or output: (the message)
  • We write that message to the console

Let's learn through code!

function getGenerationalCohort(yearBorn) {

	var generationalCohort = ""
	
	if (yearBorn > 1900 && yearBorn <= 1926) {
		generationCohort = "GI Generation";
	}
	else if (yearBorn > 1926 && yearBorn <= 1945) {
		generationalCohort = "Silent Generation";
	}
	else if (yearBorn > 1945 && yearBorn <= 1964) {
		generationalCohort = "Baby Boomers";
	} 
	else if(yearBorn > 1964 && yearBorn <= 1980) {
		generationalCohort = "Generation X";
	}
	else if(yearBorn > 1980 && yearBorn <= 2001) {
		generationalCohort = "Millenium";
	}
	else if(yearBorn > 2001 && yearBorn < 2020) {
		generationalCohort = "Generation Z";
	}
	else {
		generationalCohort = "Outside of our named generations";
	}

	return generationalCohort;
}

var year = 1985;
var cohort = getGenerationalCohort(year);
console.log("The generational cohort of someone born in " + year + " is: " + cohort + ".");
